Théorie ZM : Formalisation, Simulation et Validation

Site web : https://zm-theory.github.io/#publications

Contexte

La Théorie ZM propose une ontologie informationnelle où espace, temps, matière, énergie et conscience émergent d’un réseau fractal de pixels d’information pondérés.
L’objectif est d’unifier lois physiques fondamentales, phénomènes cosmologiques et complexité cognitive dans un cadre mathématique cohérent, falsifiable et simulable.

Objectifs

Formaliser la dynamique multi-échelle des graphes d’information pondérés.

Simuler l’émergence de structures stables, métriques effectives et signatures fractales.

Produire des prédictions testables reliées aux constantes fondamentales et signatures cosmologiques.

Regrouper publications, code et données pour assurer une reproductibilité complète.
